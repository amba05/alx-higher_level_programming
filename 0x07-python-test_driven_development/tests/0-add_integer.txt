============================
How to Use 0-add_integer.py
============================

The function is run with ``add_integer()``

Numbers
=======

``add_integer()`` returns the sum of its argument. 

::
	>>> add_integer = __import__('0-add_integer').add_integer
        

	>>> add_integer(3, 3)
        6

	>>> add_integer(1, 2)
        3

	>>> add_integer(100, -2)
        98

	>>> add_integer(100.3, -2)
        98

	>>> add_integer(-3, -2)
        -5

It can also work when only the first argument is provided
::
        >>> add_integer(6)
        104

It raises a typerror for arguments that are not int or float
:: 
        >>> add_integer('2', 98)
        Traceback (most recent call last):
        TypeError: a must be an integer

::
        >>> add_integer(5, '4')
	Traceback (most recent call last):
	...
	TypeError: b must be an integer
