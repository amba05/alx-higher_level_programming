============================
How to Use 3-say_my_name.py
============================

The function is run with ``say_my_name(first_name, last_name="")``

Array 
=======

``say_my_name()`` prints **My name is <first name> <last name>**

::
	>>> from 3-say_my_name import say_my_name
        >>> say_my_name("John", "Smith")
	My name is John Smith


It can also work for this
::
        >>> say_my_name("Walter", "White")
	My name is Walter White


It can also work for this.
:: 
        >>> say_my_name("Bob")
	My name is Bob


It raises a ``TypeError`` if first name is not a string
::
        >>> say_my_name(12, "White")
	Traceback (most recent call last):
	...
	TypeError: first_name must be a string



It raises a ``TypeError`` if div last name is not a string
::
        >>> say_my_name("Walter", 2.5)
        Traceback (most recent call last):
	...
        TypeError: last_name must be a string
